                                                                                            # RabbitMQ 说明

                                                                                            • MQ 全称为 Message Queue, 消息队列（MQ）是一种应用程序对应用程序的通信方法。应用程序通过读写出入队列的消息（针对应用程序的数据）来通信，而无需专用连接来链接它们。
                                                                                              • RabbitMQ 是一个在 AMQP 基础上完整的，可复用的企业消息系统。他遵循 Mozilla Public License 开源协议。
                                                                                            • RabbitMQ WIKI：https://zh.wikipedia.org/zh/RabbitMQopen in new window
                                                                                            • RabbitMQ 百科：http://baike.baidu.com/view/4095865.htmopen in new window
                                                                                            • RabbitMQ 官网：http://www.rabbitmq.com/open in new window
                                                                                            • RabbitMQ 官网下载：http://www.rabbitmq.com/download.htmlopen in new window
                                                                                            • RabbitMQ 官网安装文档：http://www.rabbitmq.com/install-rpm.htmlopen in new window

                                                                                            # Prometheus 安装和配置

                                                                                            • 不错的发展史说明：https://caicloud.io/blog/5a5db4203255f5063f2bd462open in new window
                                                                                            • 特别说明：一般这类环境要尽可能保证所有服务器时间一致
                                                                                            • Prometheus 本地存储不适合存长久数据，一般存储一个月就够了。要永久存储需要用到远端存储，远端存储可以用 OpenTSDB
                                                                                            • Prometheus 也不适合做日志存储，日志存储还是推荐 ELK 方案
